Overview
The Dragon Kunai asset is a high-quality 3D weapon model designed for use in Unity projects. It is supported for both Universal Render Pipeline (URP) and High Definition Render Pipeline (HDRP).

Features
Three Distinct Material Types:
	Metal – Realistic metal finish with optional details and color options.
	Jade – Mystical jade texture with optional details.
	Painted – Weathered painted surface with multiple color options.
	Bloody - Bloody Metal.
	

PBR-Ready Textures (2K Resolution)
	URP: MetallicRoughness workflow.
	HDRP: MaskMap workflow.

High-Quality Normal Maps
	Each material type includes custom Normal Maps for added surface detail.
	Metal and Jade variations feature additional Detail Normal and Roughness maps.

Multiple Albedo Options
	Metal and Paint materials come with several Albedo variations to suit different aesthetics.

Includes a lower Level of Detail (LOD) version of the Kunai Mesh
